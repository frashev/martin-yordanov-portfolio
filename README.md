# Martin Yordanov Portfolio

React 19 + React Router 7 + Tailwind CSS v4 portfolio site for Martin Yordanov,
focused on kinetic furniture, drawing machines, and functional art objects.

Live GitHub Pages URL:
https://frashev.github.io/martin-yordanov-portfolio/

## Current Content

- Home page with Martin's positioning and project imagery.
- Projects page for the kinetic sand table and wall plotter.
- Process page for build approach and placeholder biography.
- Gallery with the supplied project photos.
- Contact page using the starter-kit Supabase-ready contact service.

All biography, contact details, domain values, social links, technical specs,
and testimonials are placeholders until Martin provides verified copy.

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run test:e2e
```

## Environment

Copy `.env.example` to `.env.local` when Supabase is ready and set:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`

Do not commit `.env*` files.
